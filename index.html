Welcome to the Beginner Programming Reference!

This guide is for 1st year students learning C++, HTML, CSS, and JavaScript.

C++

Learn about basic data types and arithmetic operations. More info: W3Schools C++

Example: Declaring int and string variables in C++

#include <iostream>
#include <string>
using namespace std;

int main() {
    int age = 18;
    string name = "Alice";
    cout << "Name: " << name << endl;
    cout << "Age: " << age << endl;
    return 0;
}

Output:
    Name: Alice 
    Age: 18

Line-by-line explanation:
#include <iostream> – Includes the library for input and output.
#include <string> – Includes the library for using strings.
using namespace std; – Allows using standard library names without prefix.
int main() – The main function where the program starts.
int age = 18; – Declares an integer variable named age and sets it to 18.
string name = "Alice"; – Declares a string variable named name and sets it to "Alice".
cout << "Name: " << name << endl; – Prints "Name: Alice" to the screen.
cout << "Age: " << age << endl; – Prints "Age: 18" to the screen.
return 0; – Ends the program and returns 0 to the operating system.

HTML

Learn how to structure web pages. More info: W3Schools HTML

Example: Creating a heading and a paragraph in HTML

<!DOCTYPE html>
<html>
<body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Output:
    Hello, World!
    This is a paragraph.

Line-by-line explanation:
<!DOCTYPE html> – Declares the document type and version (HTML5).
<html> – The root element of the HTML page.
<body> – Contains the visible content of the page.
<h1>Hello, World!</h1> – Displays a large heading.
<p>This is a paragraph.</p> – Displays a paragraph of text.
</body> – Ends the body section.
</html> – Ends the HTML document.

CSS

Learn how to style web pages. More info: W3Schools CSS

Example: Styling a heading and paragraph

h1 {
    color: blue;      /* Changes the heading text color to blue */
    font-size: 32px;  /* Sets the heading font size to 32 pixels */
}
p {
    color: gray;      /* Changes the paragraph text color to gray */
    font-size: 18px;  /* Sets the paragraph font size to 18 pixels */
}

Line-by-line explanation:
h1 { color: blue; } – Sets the color of all <h1> headings to blue.
h1 { font-size: 32px; } – Makes the <h1> text larger.
p { color: gray; } – Sets the color of all <p> paragraphs to gray.
p { font-size: 18px; } – Makes the <p> text a bit larger.

This CSS changes the appearance of headings and paragraphs on your web page.

JavaScript

Learn how to add interactivity to web pages. More info: W3Schools JavaScript

Example: Declaring variables and showing an alert

let name = "Alice";
let age = 18;
alert("Name: " + name + "\nAge: " + age);

Output: 
    Name: Alice 
    Age: 18

Line-by-line explanation:
let name = "Alice"; – Declares a variable called name and assigns it the value "Alice".
let age = 18; – Declares a variable called age and assigns it the value 18.
alert("Name: " + name + "\nAge: " + age); – Shows a popup alert displaying the name and age. The \n creates a new line in the alert box.
